# Windows 本地安装程序

本目录保存 Fish-Agent Windows 本地程序的源码和打包配置。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `launcher_app.py` | V3.4.0 本地启动器与桌面 UI，负责图片选择、分析调用、结果展示、历史记录、批量分析和 PDF/JSON 导出 |
| `desktop_app.py` | 早期桌面应用参考实现，保留为本地推理核心与 UI 演进参考 |
| `Fish-Agent.spec` | PyInstaller 打包配置 |
| `Fish-Agent-installer.iss` | Inno Setup 安装包脚本 |
| `assets/fish_agent_icon.ico` | Windows 程序图标 |

## 当前 UI 优化点

- 选项卡使用高对比度选中态，避免选中标签文字隐身。
- 结果滚动区限制边界滚动，减少历史记录页的空白回弹。
- 选项卡和正文标题去除不稳定图标占位符，避免图标字体失效时暴露英文占位文本。
- 历史记录卡片左侧显示 50 x 50 图片缩略图，并增加浅色模块背景和图文间距。
- 置信度仪表盘锁定比例，数值使用固定字号，并按文本总宽度居中于半弧中央。
- 概率分布图增加图例间距、文字测量和图表留白，减少重叠。
- AI 关注区域滑块只显示当前百分比，轨道区分已选和未选区域。
- 详细分析、积极指标等列表增加垂直行距，提升阅读舒适度。
- PDF 报告中的原图和热力图按宽高比缩放，不拉伸图像证据。
- PDF 建议区使用正式项目符号列表。
- 批量分析按钮改为幽灵按钮，和普通操作区分。

## 打包提示

安装器依赖 PyInstaller 与 Inno Setup。模型权重和最终 `.exe` 体积较大，不纳入源码仓库；请在本地打包流程中把模型文件放入打包脚本指定位置。
